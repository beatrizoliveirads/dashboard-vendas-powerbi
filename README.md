# Dashboard de Vendas — Power BI

Dashboard interativo de análise de vendas desenvolvido no Power BI com dados públicos do Kaggle.

## Objetivo
Analisar o desempenho de vendas por categoria, país e período,
gerando insights para apoio à tomada de decisão.

## Indicadores do Dashboard
- Total de Vendas (KPI)
- Vendas por Categoria de Produto
- Vendas por Mês
- Vendas por País

## Ferramentas utilizadas
- Power BI Desktop
- DAX
- Power Query
- Excel (tratamento dos dados)

## Arquivos
- `dashboard-vendas-.pbix` — arquivo do dashboard Power BI
- Prints do dashboard
